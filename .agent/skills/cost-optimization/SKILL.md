# AIコスト最適化・分析スキル (cost-optimization)

このスキルは、`TokenCalc` の実行結果に基づき、AI利用コストの算出と最適化のアドバイスを提供するためのものです。

## 概要
AIエージェント（私）は、このスキルを参照することで、単なるトークン計数を超えた「コスト意識を持った開発支援」が可能になります。

## スキルの能力
- **コスト算出**: `scripts/cost_calc.py` を使用して、現在のプロジェクト全体のトークン数から最新価格に基づく見積もりを出します。
- **ボトルネック特定**: どのファイルがコストの多くを占めているかを分析します。
- **最適化のアドバイス**: コードの冗長性削減や、AIプロンプトから除外すべきファイルの提案を行います。

## ワークフロー
1. `TokenCalc` でプロジェクトをスキャンする。
2. その出力を `cost-optimization/scripts/cost_calc.py` に渡し、コストレポートを生成する。
3. エージェントはレポートを解釈し、ユーザーに最適化プランを提示する。

## ベストプラクティス
- **不要な依存関係の除外**: `.venv` や `node_modules` は必ず除外する。
- **コメント密度の調整**: 開発中は冗長なドキュメントコメントを一時的に非表示にすることで、数KB単位の節約が可能。
- **モデルの使い分け**: テスト実行は安価なモデル（例: Claude 3 Haiku / GPT-4o mini）で行うよう推奨する。
